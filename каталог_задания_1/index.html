При вызове команды ping через некоторый промежуток времени мы получаем время, за которое доходят данные от нашего ПК до сервера по данному ip-адрессу.

Из A-записей можно узнать адрес на котором находится сайт(связывает домен и ip) Из MX-записей можно узнать адресс почтового сервера и его приоритет, через который будут проходить данные для нашего сайта.

С помощью этой команды можно узнать кто является владельцем данного домена, когда он был зарегистрирован, до какого числа проплачен, кто является регистратором.
